# JBX Consórcios — CRM

Sistema de gestão de carteira com geração de propostas profissionais.

## Como publicar GRATUITAMENTE no Vercel (5 minutos)

### Passo 1 — Criar conta no GitHub
Acesse https://github.com e crie uma conta gratuita.

### Passo 2 — Criar repositório
1. Clique em **"New repository"**
2. Nome: `jbx-crm`
3. Clique em **"Create repository"**

### Passo 3 — Fazer upload dos arquivos
1. No repositório criado, clique em **"uploading an existing file"**
2. Arraste TODOS os arquivos desta pasta (inclusive a pasta `src/`)
3. Clique em **"Commit changes"**

### Passo 4 — Publicar no Vercel
1. Acesse https://vercel.com e clique em **"Sign up with GitHub"**
2. Clique em **"Add New Project"**
3. Selecione o repositório `jbx-crm`
4. Clique em **"Deploy"**
5. Aguarde ~1 minuto — seu site estará no ar!

### Resultado
Você receberá um link como: `https://jbx-crm.vercel.app`

Com o sistema hospedado:
- ✅ **🖨️ Gerar PDF** — abre a proposta e imprime como PDF
- ✅ **💬 Enviar por WA** — envia link da proposta direto pelo WhatsApp
- ✅ O cliente clica no link e vê a proposta completa no celular
- ✅ Dados salvos no navegador (localStorage)
